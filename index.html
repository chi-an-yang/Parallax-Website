視差滾動效果

By Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，讓網站背景的移動速度慢於前景，形成訪客在向下捲動頁面時的 3D 效果，增加景深並營造更沉浸的瀏覽體驗。視差源自光學錯覺，因為人眼會把近處的物體看得比遠處 的更大，於是會覺得遠處物體移動得更慢。這種錯覺早已被運用在各種媒介中，帶來更逼真的效果。 它最早出現在傳統動畫中，例如迪士尼的《白雪公主與七矮人》，以及像《超級瑪利歐》這類的遊戲。 隨著 CSS 和 HTML 的進步，視差效果才逐步發展為我們如今熟悉的網頁設計形式。

2007 年，網頁開發者 Glutnix 在部落格上分享了早期的瀏覽器視差滾動實作與示範，使用 JavaScript 和支援當時 IE6 等瀏覽器的 CSS2。直到 2011 年 HTML5 與 CSS3 技術普及後，這項技巧才真正受到 網頁設計師歡迎。擁護者認為，視差是一種擁抱網頁流動性的簡單方式，並透過視差背景來提升 使用者參與度與整體體驗。不過普渡大學 2013 年的研究指出：「雖然視差滾動增強了使用者體驗的 某些面向，但未必能提升整體使用者體驗。」


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像的兩個主要原因：

  • 更能傳達故事

    視差圖像非常適合用來說故事。好的設計重在溝通，當我們打造網站時，希望與訪客對話，並講述 產品、服務或品牌的動人故事。視差滾動能讓訪客沉浸在獨特的線上體驗中，甚至能在單頁網站內 讓訪客完整閱讀關於產品或品牌的故事，而無須跳轉多個頁面。

  • 提升使用者參與度

    視差圖像也能提升使用者參與度。降低網站跳出率是網頁設計中需要關注的目標，跳出率代表訪客 只看一頁就按下「返回」或關閉分頁的比例。高跳出率會影響搜尋排名，因此任何能降低跳出率的 作法都值得嘗試。一個設計良好的視差頁面能提供足夠的動態體驗，讓使用者停留更久。


想在網站加入視差滾動效果？請留意以下建議：

  • 適度使用

    視差滾動在特定元素上效果更強，而不是在所有頁面、所有地方都使用。可放在頁首、標題或僅限首頁， 讓視差強化設計而非分散注意力。

  • 圖像壓縮

    視差滾動會用到各式媒體與 CSS 形狀來營造景深，不希望因為過大的圖檔拖慢載入，影響效能與體驗。

  • 善用色彩

    視差滾動不僅是前景與背景的圖像，色彩同樣重要。背景可用較低飽和度的色調，前景採更鮮明的顏色， 以營造更強烈的深度感。
