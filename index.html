視差捲動效果

作者：Vineet Kumar

什麼是視差捲動？

視差捲動是一種網頁設計技巧，讓網站背景的移動速度比前景慢， 在捲動時營造 3D 視覺效果，帶來深度與沉浸式的瀏覽體驗。視差源自光學錯覺： 人眼會把近處物體看得比遠處大，因此我們會覺得遠處物體移動得更慢。 這個錯覺早已被應用在不同媒介，例如迪士尼《白雪公主與七矮人》等傳統動畫、 以及《超級瑪利歐》等電玩。隨著 CSS 與 HTML 的進步，視差效果才發展成今天的網頁設計元素。

其中一個最早的視差網頁實作是開發者 Glutnix 在 2007 年分享的示範， 以 JavaScript 與 CSS 2 製作，甚至支援 IE6。到了 2011 年， 隨著 HTML5 和 CSS3 的普及，視差效果開始在設計師間流行，被視為展現網頁流動感的簡單方式。 支持者認為視差背景能提升互動性與整體體驗；然而 2013 年普渡大學的研究指出： 「雖然視差捲動提升了部分使用者體驗，但並不一定能改善整體體驗。」


為什麼要使用視差圖形？

    設計師應嘗試視差圖形的兩個主要原因：

  • 有助於說故事

    視差圖形非常適合敘事。好的設計就是良好的溝通—在網站中， 我們想和訪客對話，講出產品、服務或品牌的精彩故事。 視差捲動能讓訪客沉浸在獨特的線上體驗，甚至可以打造一頁式網站， 讓使用者不需跳轉就能閱讀完整故事。

  • 有助於提升互動

    視差圖形可以提升互動並降低跳出率， 也就是訪客在瀏覽單一頁面後就按下「返回」或關閉分頁的比例。 高跳出率會傷害搜尋排名，因此任何能讓使用者停留的方式都值得嘗試。 精心打造的視差頁面能提供動態體驗，通常能讓使用者停留更久。


想在網站加入視差效果時，可以參考以下小撇步：

  • 有選擇地使用

    視差捲動運用在特定元素上更有力量，例如標題或首頁， 而非每一頁都使用。它應該是強化設計，而不是分散注意力。

  • 注意影像壓縮

    視差需要多種媒體與 CSS 形狀來營造深度， 別讓過大的圖片拖慢網站效能，破壞使用者體驗。

  • 善用色彩

    視差不只關乎前後景的圖片，也和色彩有關。 你可以在背景使用較低飽和度的色調，前景用更活潑的顏色， 來創造更強烈的層次感。
